Using Of The Package
====================
1. Attach QMazeEngine script to the game object.
2. Specify the necessary parameters in the QMazeEngine script:
- Maze size
- Maze pieces size
- Maze pieces pack
- Creation script of maze geometry
- Count and, if necessary, positions of the start and finish maze pieces. Also you can specify entrances of the maze.
- Seed, if necessary 
3. Specify the necessary parameters in the QMazePiecePack script:
- Specify the game objects appropriate to pieces.
- If necessary, several game objects can be set for each piece. Click plus/minus to add/remove it.
- Frequency of occurrence can be specified for some pieces.
4. Click the "Generate a Maze" button to create a maze in the edit-time mode or call "generateMaze" function from code (see the example scene).
